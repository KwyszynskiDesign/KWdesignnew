KW Buttons — system przycisków · v1.0 (żywa dokumentacja)

Wersja v1.0 — zatwierdzona 2026-07-11. Jedyny obowiązujący styl systemu .kw-btn. Źródło prawdy: assets/css/kw-buttons.css.

  • Paleta: głęboki teal (primary #10809d → #0c6076, akcent #0e7490) — premium, nie „SaaS cyan".
  • Radius: ~11px (miękkie krawędzie). Typy: primary / secondary / tertiary / icon. Rozmiary: sm / md / lg.
  • Stany: hover / active / focus-visible / disabled.
  • Hover: lift −2px + głębszy, miękki cień; subtelny highlight górnej krawędzi.
  • Poświata: jasny cyjan „zapalający się" ze środka (0.70) — primary, secondary, icon.
  • Rozbłysk tekstu: delikatne podświetlenie napisu na hover (typy wypełnione).
  • Wariant kontekstowy --inverse (v1.1): CTA na ciemnym/teal tle — biały fill + teal tekst.
